# Pedidos Epuyen - Operación Solidaria

Formulario web para tomar pedidos de pizza solidaria en la ciudad de Epuyen. Los datos se envían automáticamente a una planilla de Google Sheets vía Google Apps Script.

## Stack

- HTML5 + JavaScript vanilla
- Tailwind CSS (CDN)
- Google Apps Script (backend)
- Google Sheets (base de datos)

## Funcionalidades

- Formulario de pedido con validación nativa
- Envío automático a Google Sheets
- Fecha y hora registradas automáticamente
- Diseño mobile-first optimizado para uso en campo
